TAAS - (Template As A Service)

The new way to start your projects.

Today, we start tons of projects, but we waste a lot of time preparing the environment, copying files, etc.Imagine if it were possible to automate this thnaks to a little file that would do all this boring tasks for us.

T3mplt

Thanks to T3mplt it is now possible to create projects from your own templates. T3mplt works with all languages, all file types and all projects. It is very easy to use and its CLI allows you to integrate it in your workflows.

Made by professionals for professionals.
